<h2>Features</h2>
<ul>
    <li>Clean and intuitive chat interface</li>
    <li>Message input field with send button</li>
    <li>Display messages in a conversation format</li>
    <li>Differentiate between sent and received messages</li>
    <li>Real-time message display</li>
    <li>Smooth scrolling chat view</li>
</ul>

<h2>Getting Started</h2>
<p>To run this project locally:</p>

<ol>
    <li>
        <strong>Clone the repository:</strong>
        <pre><code>git clone https://github.com/Iftikhar-Shams-Niloy/flutter_simple_chat.git</code></pre>
    </li>
    <li>
        <strong>Navigate to the project directory:</strong>
        <pre><code>cd flutter_simple_chat</code></pre>
    </li>
    <li>
        <strong>Get the dependencies:</strong>
        <pre><code>flutter pub get</code></pre>
    </li>
    <li>
        <strong>Run the application:</strong>
        <pre><code>flutter run</code></pre>
    </li>
</ol>

<hr>

<h2>Requirements</h2>
<ul>

<h2>Learning Outcome</h2>
<p>This project helps in understanding:</p>
<ul>
    <li>How to build a chat UI layout in Flutter</li>
    <li>Working with <code>ListView</code> and <code>ScrollController</code> for message display</li>
    <li>Implementing <code>TextField</code> for user input</li>
    <li>Managing chat state with <code>StatefulWidget</code></li>
    <li>Differentiating UI elements (sent vs received messages)</li>
    <li>Using <code>Column</code>, <code>Row</code>, and other layout widgets effectively</li>
    <li>Basic styling with containers and decorations</li>
</ul>
